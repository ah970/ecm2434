Frequently Asked Questions

Some of the most frequently asked questions that were given from our beta and alpha testers of the game!

  • Does the app store my gps data location?

    No the application does not, it uses the current user location locally on the device to calculate the distance of the player to the desintation location.

  • Is there mobile support?

    There is mobile support for these pages however depending on your device we reccomend you to play it on a larger screen such as a tablet or a large monitor.

  • Is the application open source?

    Yes the entire project is open sourced and can be found on github. Feel free to contribute following the procedures found on the readme page on the github repository.
