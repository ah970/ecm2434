About Us

This project was made for the Group Software Engineering Project in which we have developed an app using the latest geolocation API technology to make a fun and exciting Hot and Cold GPS location based game!

  • Latest GPS Technology

Frequently Asked Questions

Here are some of the most frequently asked questions that were asked by our beta and alpha testers of the game!

  • Does the app store my GPS data location?

    No, the application does not. It uses the current user location locally on the device to calculate the distance of the player to the destination location.

  • Is there mobile support?

    There is mobile support for these pages, however depending on your device we reccomend you to play it on a larger screen such as a tablet or monitor.

  • Is the application open source?

    Yes, the entire project is open source and can be found on GitHub. Feel free to contribute following the procedures found on the README page on the GitHub repository.
